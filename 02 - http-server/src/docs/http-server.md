## Criando um servidor HTTP com Node.js

Neste estudo foi criado um servidor HTTP utilizando apenas o módulo nativo `http` do Node.js.

### Objetivo

Entender como o Node.js lida diretamente com requisições e respostas HTTP, sem o uso de frameworks como Express.

### Conceitos abordados

- Módulo `http`
- `http.createServer`
- Objetos `req` e `res`
- Status code e headers
- Finalização da resposta com `res.end()`

### Como executar

```bash
node index.js
```
