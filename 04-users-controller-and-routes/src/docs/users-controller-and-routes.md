# Controller de Usuários e Sistema de Rotas

Este estudo demonstra a criação de um controller de usuários e um sistema de
rotas simples utilizando Node.js puro, sem frameworks.

## Conceitos abordados

- Separação de responsabilidades
- Controllers para lidar com requisições
- Sistema de rotas baseado em array
- Dispatcher de rotas no servidor HTTP
- Uso de mocks para simular dados

## Estrutura

- controllers: lógica de resposta
- routes: definição das rotas da aplicação
- mocks: dados simulados
- server.js: ponto de entrada da API

## Rota disponível

### GET /users

Retorna uma lista de usuários em formato JSON.
