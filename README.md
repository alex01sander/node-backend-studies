# Estudos de Backend

Repositório para organizar meus estudos de **backend**, de forma prática e constante.  
A ideia é evoluir aos poucos, entendendo os fundamentos e aplicando tudo em código.

Sem copiar e colar sem saber o que está acontecendo.

---

## Objetivo

- Criar uma base sólida em backend
- Trabalhar com Node.js e ecossistema
- Entender como APIs e serviços funcionam
- Praticar lógica, arquitetura e boas práticas
- Manter consistência de estudo e commits

---

## Forma de estudo

Os estudos são organizados por dias.

Em cada dia:
- Estudo o conceito
- Aplico na prática
- Versiono o que foi feito

Mesmo coisas simples fazem parte do processo.

---

## Commits

Os commits refletem o aprendizado real, não quantidade.

Exemplos:
```bash
feat: implementação de leitura de arquivos
refactor: melhoria no fluxo assíncrono
chore: ajustes iniciais

Observação

Este repositório é apenas um registro de estudo e evolução em backend.
Não representa um projeto final ou produto pronto.
